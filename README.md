# dog-fetcher-sdk
sdk for fetching random dog images from the internet


Requirements: 
---------------------------------------------
* Swift 5.5+
* iOS 16.0+


Installation:
---------------------------------------------
CocoaPods

source 'https://github.com/CocoaPods/Specs.git'

platform :ios, '16.0'

use_modular_headers!

target 'MyApp' do

  pod 'DogFetcher'
  
end
